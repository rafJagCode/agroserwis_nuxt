## Build Setup

- Clone repository

  ```sh
  git clone https://github.com/rafJagCode/hervestsolutions.git
  ```

- Install dependencies

  ```sh
  cd hervestsolutions
  yarn install
  ```

- Serve with hot reload at localhost:3000

  ```sh
  yarn dev
  ```

- Build for production and launch server

  ```sh
  yarn build
  ```

  ```sh
  yarn start
  ```

- Generate static project

  ```sh
  yarn generate
  ```
